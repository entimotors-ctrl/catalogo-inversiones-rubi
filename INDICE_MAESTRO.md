## 🔧 CUSTOMIZACIONES COMUNES

### Cambiar Número de WhatsApp
**Archivo:** `CatalogoPublico.jsx` línea 8
```javascript
const numeroWhatsApp = "50499999999" // ← Tu número
```

### Cambiar Color Primario
**Archivo:** `App.css` línea 14
```css
:root {
  --color-primary: #002855; /* Tu color azul */
}
```

### Cambiar Color Verde WhatsApp
**Archivo:** `App.css` línea 16
```css
:root {
  --color-success: #25D366; /* Tu color verde */
}
```

### Cambiar Número de Columnas
**Archivo:** `CatalogoPublico.jsx` línea 65 (Grid)
```jsx
<div className="grid [...] lg:grid-cols-4 [...] gap-6">
                                     ^^^
                    Cambia a: 3, 5, 6, etc.
```

### Ajustar Tipografía
**Archivo:** `App.css` línea 1-3 (Google Fonts)
```css
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;600;700&display=swap');
```

## ⚠️ TROUBLESHOOTING RÁPIDO

| Problema | Solución |
|----------|----------|
| CSS no carga | Verifica `import './App.css'` en App.jsx |
| Búsqueda no funciona | Recarga la página, verifica console |
| Categorías no filtran | Verifica que fetch de `/categorias` funciona |
| WhatsApp no abre | Verifica formato de número: sin espacios, con código país |
| Mobile se ve mal | Abre DevTools, toggle device toolbar |
| Animaciones lentissimas | Deshabilita otras extensiones del navegador |

Más detalles en: [QUICK_START.md](QUICK_START.md#-posibles-issues-y-soluciones)
